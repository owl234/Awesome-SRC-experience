# 🛡️ SRC 攻防实战经验库 (Target-Centric Playbooks)

> [!IMPORTANT]
> **探索路线**: 本部分按照**目标类型 (Target Type)** 进行分类映射：物联网 (IoT)、Web 应用、云原生 (Cloud) 以及基础设施 (Infrastructure)。每篇 Playbook 均遵循“背景 -> 攻坚 -> 自动化”的职业级闭环。

## 💡 核心攻坚领域

这里的每一篇文章都是从上千个 403、500 报错中提炼出的实战精华：

- **📡 IoT & 终端**: 攻破硬件边界，深入固件与专有协议。
- **🌐 Web & 业务**: 403 绕过、逻辑缺陷与现代 WAF 下的 Payload 进化。
- **☁️ Cloud & 云原生**: 元数据、S3 桶与云厂商特有的攻击向量。
- **🏰 Infra & 架构**: Web 服务器缺陷、DNS 劫持与内网横向基础。

## 🏆 白帽进阶指南：如何成为 P1/P0 捕手

1.  **资产深挖**: 国内优先攻击**核心资产**（边缘资产收益极低）。优先测试中高危，通过低危漏洞建立信心。
2.  **逆向思维**: 始终以**开发者的视角**思考：如果是我想实现这个功能，我会如何写代码？我会漏掉什么样的校验逻辑？
3.  **思路闭环**: 
    - 🔍 发现点位 (Discovery)
    - 🧪 复现验证 (Reproduction)
    - 📜 沉淀思路 (Summarization)
    - 🤖 自动化 (Automation)
4.  **执着与身体**: 兴趣是第一动力，但在卷的同时，请务必保证休息。
