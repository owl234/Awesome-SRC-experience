# 🛡️ SRC 攻防实战经验库 (Security Playbooks)

> [!IMPORTANT]
> **探索路线**: 本部分按照**漏洞类型**与**攻击向量**进行模块化。每篇 Playbook 均遵循“漏洞原理 -> 攻坚重难点 -> 自动化思路”的职业级闭环。

## 💡 核心攻坚战术

这里的每一篇文章都是从上千个 403、500 报错中提炼出的财富：

- **权限绕过**: 403 不代表结束，往往是精彩的开始。
- **逻辑缺陷**: 业务逻辑是人类思维的投影，脆弱而多变。
- **注入进阶**: 现代 WAF 下的 Payload 进化论。

## 🏆 白帽进阶指南：如何成为 P1/P0 捕手

1.  **资产深挖**: 国内优先攻击**核心资产**（边缘资产收益极低）。优先测试中高危，通过低危漏洞建立信心。
2.  **逆向思维**: 始终以**开发者的视角**思考：如果是我想实现这个功能，我会如何写代码？我会漏掉什么样的校验逻辑？
3.  **思路闭环**: 
    - 🔍 发现点位 (Discovery)
    - 🧪 复现验证 (Reproduction)
    - 📜 沉淀思路 (Summarization)
    - 🤖 自动化 (Automation)
4.  **执着与身体**: 兴趣是第一动力，但在卷的同时，请务必保证休息。
